# Novel Poster Example

一个 Codex 海报生成样例：从本地小说文本处理、角色上下文检索、人设 prompt 整理，到生成「薰儿」成熟期与青涩期两张 9:16 海报。

## 打开样例

直接打开 [`codex海报生成样例.html`](codex海报生成样例.html)。

## 仓库内容

- `codex海报生成样例.html`：完整流程展示页
- `薰儿-txt/character-brief.md`：角色人设整理
- `薰儿-txt/poster-prompt.txt`：成熟期海报 prompt
- `薰儿-txt/poster-prompt-young-xiao-xuner.txt`：青涩期海报 prompt
- `薰儿-txt/*poster-9x16-4k.png`：两张 2160x3840 海报
- `薰儿-txt/summary.json`、`薰儿-txt/top-chapters.txt`：检索统计与章节命中摘要
